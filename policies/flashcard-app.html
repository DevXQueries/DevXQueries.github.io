FlashCard App

A powerful memorization tool built with Jetpack Compose for modern UI/UX.

Project Overview

FlashCard App is designed to help users memorize information efficiently. Built natively for Android using Jetpack Compose, it offers a smooth and responsive interface. The app features secure authentication via Firebase and integrates AdMob for monetization while maintaining a user-friendly experience.

Key Features

  • Modern UI built with Jetpack Compose
  • Secure user authentication with Firebase
  • Cloud sync for flashcard decks
  • Spaced repetition algorithm
  • Ad-supported (AdMob integration)
