Nexus Infrastructure

IaC visualization tool that maps Terraform configurations to visual diagrams.

Project Overview

Nexus Infrastructure bridges the gap between code and architecture. By parsing Terraform state files, it generates interactive, auto-updating diagrams of your cloud infrastructure. This tool helps DevOps teams validate architecture, identify orphaned resources, and generate documentation automatically.

Key Features

  • Automatic diagram generation from .tfstate
  • Dependency graph visualization
  • Cost estimation overlay
  • Drift detection alerts
  • Export to SVG, PNG, and PDF

Technologies

GoVue.jsTerraformD3.js
